# Simple

## 信息

| 字段 | 值 |
|---|---|
| 目录名 | `simple` |
| 版本 | `1.1.0` |

| 模板引擎 | `ejs` |

## 自定义参数

在应用「主题 → 自定义」里可以配置以下选项：

### 布局

| 参数 | 说明 | 默认值 |
|---|---|---|
| 菜单栏宽度 | 可填像素类型（如：320px）或百分比类型（如：38.2%） | `320px` |
| 封面图圆角 | 像素类型（如：3px） | `3px` |

### 渲染

| 参数 | 说明 | 默认值 |
|---|---|---|
| 是否渲染 KaTeX 公式 | - | `false` |
| 是否渲染代码高亮 | - | `false` |

### 社交

| 参数 | 说明 | 默认值 |
|---|---|---|
| Github | 链接地址 | `` |
| Twitter | 链接地址 | `` |
| 微博 | 链接地址 | `` |
| 知乎 | 链接地址 | `` |
| Facebook | 链接地址 | `` |

### 自定义样式

| 参数 | 说明 | 默认值 |
|---|---|---|
| 自定义CSS |  | `` |

### 谷歌统计

| 参数 | 说明 | 默认值 |
|---|---|---|
| 跟踪 ID | UA-xxxxxxxxx-x | `` |

### 颜色

| 参数 | 说明 | 默认值 |
|---|---|---|
| 菜单颜色 | 颜色字符串:（如：#EEEEEE、rgba(255, 255, 255, 0.9)） | `#dee2e6` |
| 内容区背景色 | 颜色字符串:（如：#EEEEEE、rgba(255, 255, 255, 0.9)） | `#f8f9fa` |


## 使用

1. 在 Gridea Pro 应用内「主题」页选择本主题
2. 在「自定义」里按需调整参数
3. 预览、发布
