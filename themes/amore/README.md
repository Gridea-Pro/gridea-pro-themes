# Amore

> 极简博客主题，聚焦阅读体验，支持明/暗双主题、阅读进度、闪念热力图等。

![preview](./assets/media/preview.png)

## 信息

| 字段 | 值 |
|---|---|
| 目录名 | `amore` |
| 版本 | `1.1.0` |
| 作者 | Amore |
| 模板引擎 | `ejs` |

## 自定义参数

在应用「主题 → 自定义」里可以配置以下选项：

### 基础设置

| 参数 | 说明 | 默认值 |
|---|---|---|
| 站点 Logo | 首页头像/站点 Logo，推荐正方形 | `` |
| 站点图标 Favicon | 浏览器标签图标，推荐 .ico 或 .png | `` |
| 页脚信息 | 支持 HTML，显示在页脚 | `` |

### 外观设置

| 参数 | 说明 | 默认值 |
|---|---|---|
| 默认主题 | 访客首次访问时默认显示的主题；之后由访客自行切换并记住其选择 | `dark` |
| 主题强调色 | 默认薄荷绿；影响链接、按钮、强调条等 | `#28df99` |

### 社交链接

| 参数 | 说明 | 默认值 |
|---|---|---|
| GitHub | 完整链接地址 | `` |
| Twitter | 完整链接地址 | `` |
| 微博 | 完整链接地址 | `` |
| 知乎 | 完整链接地址 | `` |
| Facebook | 完整链接地址 | `` |

### 统计分析

| 参数 | 说明 | 默认值 |
|---|---|---|
| Google Analytics 跟踪 ID | 例如 UA-xxxxxxxxx-x 或 G-XXXXXXXX | `` |

### 首页信息

| 参数 | 说明 | 默认值 |
|---|---|---|
| 地理位置 | 显示在站点简介下方 | `中国 · 深圳` |
| 格言内容 | 最顶级的人才喜欢一起工作，而且他们是不能容忍平庸作品的。 | `` |
| 格言作者 | 史蒂夫 · 乔布斯 | `` |

### 高级设置

| 参数 | 说明 | 默认值 |
|---|---|---|
| 自定义 CSS | 追加到 <head> 中，覆盖主题默认样式 | `` |
| 自定义 JavaScript | 追加到页面底部 | `` |


## 使用

1. 在 Gridea Pro 应用内「主题」页选择本主题
2. 在「自定义」里按需调整参数
3. 预览、发布
