## 📦 如何使用主题

1. 打开 Gridea Pro 应用，进入「主题」页面
2. 从本仓库下载想用的主题目录（`themes/<主题名>/`）
3. 放到你本地站点目录的 `themes/` 下
4. 在应用里选择它、自定义参数、发布

> 未来我们会在应用内集成一键安装，当前先走手动。

## 🙌 如何贡献

我们接受三种贡献：

| 类型 | 走 | 说明 |
|---|---|---|
| **提交新主题** | Pull Request | 在 `themes/` 下新建一个目录，提 PR |
| **更新已有主题** | Pull Request | 仅限原作者（或得到原作者同意） |
| **反馈主题问题** | Issue | 选「主题 Bug」「求主题」或「讨论」模板 |

👉 详细规则请看 [CONTRIBUTING.md](./CONTRIBUTING.md)
👉 主题目录规范看 [docs/theme-spec.md](./docs/theme-spec.md)
👉 5 分钟上手看 [docs/quickstart.md](./docs/quickstart.md)

> 我们欢迎**不完美**的主题。审美、代码、文案都可以后续迭代。只要能跑、能看，就大胆提上来。
